# Data Types & Keywords

## Data Types

Python has **5 core data types**:

| Type | Example | Description |
|------|---------|-------------|
| `int` | `42` | Whole numbers |
| `str` | `"hello"` | Text |
| `float` | `3.14` | Decimal numbers |
| `bool` | `True` / `False` | Boolean values |
| `NoneType` | `None` | Represents no value |

## Keywords

Python has reserved keywords — **you cannot use them as variable names**.

!!! warning "Reserved Keywords"
    You will get a `SyntaxError` if you try to name a variable the same as a keyword.

Examples:

```python
if    else    elif    for    while
def   class   return  import True
False None    and     or     not
in    is      pass    break  continue
```
